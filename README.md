# Ever Wall Posts

Simple wall posts on Everscale Blockchain without using smart contracts

All posts are stored in the blockchain messages and can be accessed by anyone.

Benefits:

- No storage fees
- No censorship
- Proof of ownership

This repo is only prototype.

```
npm install
```
